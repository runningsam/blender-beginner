这本书能帮你什么？

大家好！我是 Sandra Kolar，你们可能在社交媒体上知道我叫 SaTales。

我最初发布 Blender 教程、技巧和窍门只是为了好玩，但当发现大家这么喜欢这些内容后，我决定更进一步，为所有初学者写一本真正的指南。

我经常收到消息和评论，说对于那些刚开始学习 3D建模、第一次打开 Blender 的人来说，教程真的不多。

我也曾经是个初学者，我知道在没有任何基础知识的情况下学习 3D有多难。

这本电子书是为所有想学 Blender 但不知道从何开始的人准备的。

希望大家能享受和我一起学习的过程，也希望这本书能在你们的 3D学习之旅中有所帮助。

现在，让我们开始上课吧！

祝大家学习愉快！
