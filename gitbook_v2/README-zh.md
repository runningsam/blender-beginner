这本 PDF 指南能怎么帮你？

大家好！我是 Sandra Kolar，或者你们可能在我的其他社交媒体上认识我——SaTales。

我开始发布 YouTube Blender 教程、技巧和小窍门只是为了好玩，但当我意识到人们有多么喜欢它们时，我决定把这一切提升到一个新的水平，为所有初学者写一本真正的指南。

我经常收到消息和评论，说对于那些刚开始 3D 建模之旅、第一次打开 Blender 的人来说，教程并不多。

我也曾经是个初学者，我知道在没有任何先前知识的情况下学习 3D 是多么困难。

所以这本电子书是为所有想学习 Blender 但不知道怎么开始、从哪里开始的人准备的。

我希望你们会喜欢和我一起学习，希望这本书能在你们的 3D 之旅中帮助到你们。

现在，是时候开始我们的课程了！

祝大家玩得开心！
