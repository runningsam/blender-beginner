Beginners guide to Blender

大家好！我是 SaTales，今天又到了新的 Blender 课程时间啦！

如果你还没听说的话，Blender 5.0 已经发布啦！

你可以从 Blender 官方网站 Blender.org 下载。

这个新版本的 Blender 有很多很棒的新功能。我想把所有功能都展示给大家看，不过现在我先决定介绍我最喜欢的一个功能 - 圆形阵列。

那我们就开始吧！

（完整视频在我的 YouTube 频道：https://youtu.be/2b3mqOPaAZQ）

首先，选中摄像机和灯光，然后用 "X" 删除它们。

![page_327_img_1.png](images/book2_page_327_img_1.png)

用鼠标左键（LMB）选中立方体，然后进入修改器面板。

![page_327_img_2.png](images/book2_page_327_img_2.png)

进入 *`modifier → Add → Generate → Subdivision`* 表面细分。

![page_328_img_1.png](images/book2_page_328_img_1.png)

把视口层级和渲染层级都改成 3。

![page_328_img_2.png](images/book2_page_328_img_2.png)

点击鼠标右键（RMB），选择着色平滑。

![page_328_img_3.png](images/book2_page_328_img_3.png)

按 "TAB" 切换到编辑模式。

![page_329_img_1.png](images/book2_page_329_img_1.png)

用 "`S+Z`" 缩放，大概缩放到 0.25 左右。

![page_329_img_2.png](images/book2_page_329_img_2.png)

用 "CTRL+R" 添加一个环切。

![page_329_img_3.png](images/book2_page_329_img_3.png)

按数字键 1 切换到顶点选择模式，选中这两个顶点。

![page_330_img_1.png](images/book2_page_330_img_1.png)

用 "`G+Y`" 移动，大概移动 1.5 左右。

![page_330_img_2.png](images/book2_page_330_img_2.png)

用 "CTRL+R" 再添加一个新环切。

![page_330_img_3.png](images/book2_page_330_img_3.png)

再添加一个环切。

![page_331_img_1.png](images/book2_page_331_img_1.png)

按数字键 2 切换到边选择模式，选中这两个环切。

![page_331_img_2.png](images/book2_page_331_img_2.png)

用 "`G+Z`" 往上移动，大概 0.4 左右。

![page_331_img_3.png](images/book2_page_331_img_3.png)

Beginners guide to Blender

按数字键 2 切换到边选择模式。选中这两条边。

![page_332_img_1.png](images/book2_page_332_img_1.png)

用 "`G+Y`" 往左移动，大概 0.6 左右。

![page_332_img_2.png](images/book2_page_332_img_2.png)

按 "TAB" 切换回物体模式。

![page_332_img_3.png](images/book2_page_332_img_3.png)

进入 *`modifier → Add Modifier → Array`*（不要选 Array Legacy，那个是旧版的）。

![page_333_img_1.png](images/book2_page_333_img_1.png)

![page_333_img_2.png](images/book2_page_333_img_2.png)

把形状改成圆形（Circle）。

![page_333_img_3.png](images/book2_page_333_img_3.png)

把半径改成大概 0.95 左右。

![page_334_img_1.png](images/book2_page_334_img_1.png)

然后把数量改成 6（或者任何你想要的数字）。

![page_334_img_2.png](images/book2_page_334_img_2.png)

现在再调整一下半径。我把它改成了 1.38。

![page_334_img_3.png](images/book2_page_334_img_3.png)

如果你觉得花朵现在的样子不太满意，还可以继续调整数量和半径。最后，我决定把数量改成 9，半径设为 1.84。

![page_335_img_1.png](images/book2_page_335_img_1.png)

进入 *`Add → Mesh → UV Sphere`* UV 球体。

![page_335_img_2.png](images/book2_page_335_img_2.png)

用 "S" 缩放到你觉得合适的大小。点击鼠标右键（RMB），选择着色平滑。

![page_336_img_1.png](images/book2_page_336_img_1.png)

用 "`S+Z`" 缩放，调整到你觉得看起来最好的样子。

![page_336_img_2.png](images/book2_page_336_img_2.png)

然后用 "`G+Z`" 往上移动一点。按 "CTRL+A" 应用缩放。
