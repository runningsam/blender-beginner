# 第7章：向场景添加物体

当你打开 Blender 时，总有一个默认立方体等着你。

有些人会删掉那个立方体再添加一个新的，有些人会用那个立方体做场景，还有些人会删掉立方体添加其他物体。

不管怎样，你都需要知道怎么添加新物体，因为迟早你需要除立方体以外的其他东西。

那么，怎么添加新物体呢？

有几种方法。

第一种不用任何快捷键。

直接去 Add —> Mesh —> 选择你想添加的任何物体

（平面、立方体、圆、UV球体、Ico球体、圆柱、圆锥、圆环……）。

![page_44_img_1.png](images/page_44_img_1.png)

第二种是用快捷键（试着记住这个方法，建模会更轻松快捷）。

只要按"SHIFT+A"，你会看到这个菜单。选择 Mesh，然后添加你想要的物体。
