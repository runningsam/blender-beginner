# 第14章：你的第一个 3D模型

我在想给你理解 Blender 建模最好的第一个模型是什么，决定从棋子兵开始。

最后看到自己能建模出来时，既有趣又满足。

让我们开始吧！

首先，我们找一张参考图。

我在 Pixabay.com 上找到的。（https://pixabay.com/vectors/chess-game-pawn-piece-recreation-26779/）。

你可以下载这张或任何你想要的图片，只要确保是正面视图。我会在描述里放上图片下载链接。

![page_92_img_1.png](/images/page_92_img_1.png)

现在来学习如何用图片参考建模。我们目前在物体模式。

首先，用"X"删除场景里的所有东西。

![page_92_img_2.png](/images/page_92_img_2.png)

然后，点击 gizmo 上的"-Y"，这样从一开始就能正确定位。-Y 是正面。

![page_93_img_1.png](/images/page_93_img_1.png)

然后去 `Add → Image → Reference`。

之后会打开新窗口，你找到刚才下载的图片。

![page_93_img_2.png](/images/page_93_img_2.png)

选择图片，点击"Add Image"按钮。

![page_94_img_1.png](/images/page_94_img_1.png)

图片添加好了。

![page_94_img_2.png](/images/page_94_img_2.png)

右边你会看到很多工具，但现在对我们重要的是 `Data → Object` 数据属性。

![page_94_img_3.png](/images/page_94_img_3.png)

这里你可以改变图片的大小、偏移、深度、边和透明度。

我们要把 Depth 改成 Back。

![page_95_img_1.png](/images/page_95_img_1.png)

下一步是把兵放在中心，让兵的底部和 X轴对齐，方便建模。

因为参考图已经在中间了，我只要用"`G+Z`"把它往上移大约 2.29，让底部和 X轴对齐。

![page_95_img_2.png](/images/page_95_img_2.png)

添加物体之前，我们先想想根据兵的形状，哪个物体最适合建模。

我们有：平面、立方体、圆、UV球体、Ico球体、圆柱、圆锥和圆环。

最相似的是圆和圆柱。

你可以选任意一个。我用圆开始。

去 `Add → Mesh → Circle`
